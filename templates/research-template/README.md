# Šablon za istraživanja

Ovaj folder služi kao šablon za istraživačke projekte i analize.

## Struktura istraživanja

```
research-name/
├── README.md           # Hipoteza i ciljevi
├── methodology/        # Metodologija istraživanja
├── data/               # Sirovi podaci
├── analysis/           # Analize i obrada
├── findings/           # Nalazi i zaključci
└── references/         # Reference i izvori
```

## Kako koristiti

1. Definiši hipotezu ili istraživačko pitanje
2. Dokumentuj metodologiju pre početka
3. Čuvaj sirove podatke odvojeno od obrađenih
4. Beleži sve korake analize za reproduktivnost

## Konvencije

- Svaki nalaz mora imati izvor
- Verzionisanje podataka je obavezno
- Zaključci moraju biti potkrepljeni podacima
