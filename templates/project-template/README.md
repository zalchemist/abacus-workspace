# Šablon za nove projekte

Ovaj folder služi kao šablon za kreiranje novih projekata u abacus-workspace.

## Struktura projekta

```
project-name/
├── README.md           # Opis projekta
├── docs/               # Dokumentacija
├── data/               # Podaci (input/output)
├── scripts/            # Skripte i alati
└── output/             # Rezultati i izveštaji
```

## Kako koristiti

1. Kopiraj ovaj folder sa novim imenom projekta
2. Ažuriraj README.md sa opisom projekta
3. Dodaj potrebne podfoldere prema potrebi
4. Registruj projekat u glavnom STATUS.md

## Konvencije

- Nazivi foldera: kebab-case (npr. `moj-projekat`)
- Dokumentacija: Markdown format
- Commit poruke: srpski jezik
